En el ejemplo que sigue, el orden natural es:

"aa" < "ah" < "ai"

fun main() {
 // Menor o igual que (<=)
 println("ah" <= "aa"); //F
 println("ah" <= "ah"); //T
 println("ah" <= "ai"); //T
 // Mayor o igual que (>=)
 println("ah" >= "aa"); //T
 println("ah" >= "ah"); //T
 println("ah" >= "ai"); //F
}
